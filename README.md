# FEMA R8 EVR Tool - 2D SID 128 Check

The purpose of this github is to provide the arcpy toolbox and documentation to assist mapping partners in the development of mapping products for 2D modeled floodplains that meet FEMA Working Standard ID 128 (SID 128). Please refer to the [Wiki](https://github.com/Dewberry/r8-2d-flood-profiles/wiki/1.0-OVERVIEW) or the [PDF Documentation](https://github.com/Dewberry/r8-2d-flood-profiles/blob/main/docs/R8_Quick_Guide_2D-Profiles.pdf) for more background and how to use this tool.
